## 执行的查询语句

- 找到子类别的关系

```
SELECT ?subject ?object
	WHERE { ?subject rdfs:subClassOf ?object }
```

返回到匹配 rdfs:subClassOf 的125条数据

- 找到用户和对应的价格

```
SELECT ?u ?p
WHERE {
    ?s <https://www.gsmarena.com/ontologies/mobile.owl#hasPrice> ?p.
    ?u <https://www.gsmarena.com/ontologies/mobile.owl#uses> ?s.
}
```
